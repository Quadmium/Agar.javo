# Agar.javo

A Java implementation of Agar.io.

## Installation

Build project with BlueJ or another Java compiler. To run, load the MainMenu class.

## Usage

From the main menu, instances of the client and the server can be launched.
To add a server, add the IP to the bottom of the client window, click the button "Add" to the right, and then select it from the list.
To join a server, enter a valid name (or no name) and click connect after choosing one from the server list.
